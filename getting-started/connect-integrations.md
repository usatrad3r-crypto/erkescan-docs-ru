# Подключение Telegram и Discord

ErkeScan интегрируется с Telegram и Discord, чтобы вы получали алерты и управляли аккаунтом прямо из мессенджеров. В этом руководстве показано, как подключить обе платформы.

## Подключение Telegram бота

### Что вы получите

- Алерты в реальном времени из ваших кастомных фильтров
- Команды для управления подпиской
- Быстрый доступ к данным о парах
- Оценки сигналов прямо в Telegram

### Шаги подключения

**Шаг 1: Откройте бот для алертов**

Откройте [@erkescanalert\_bot](https://t.me/erkescanalert_bot) в Telegram и отправьте `/start`. Этот бот будет доставлять ваши кастомные алерты.

**Шаг 2: Откройте бот для подписки**

Откройте [@erkescan\_premium\_bot](https://t.me/erkescan_premium_bot) и отправьте `/start`. Этот бот нужен для управления подпиской и аккаунтом.

**Шаг 3: Готово**

Вернитесь в Telegram. Теперь вы начнёте получать алерты на указанный номер телефона/аккаунт.

### Основные команды Telegram бота

| Команда | Описание |
|---------|---------|
| **/start** | Главное меню и авторизация |
| **/alerts** | Список активных алертов |
| **/signals** | Последние сигналы |
| **/screener** | Быстрый доступ к скринеру |
| **/settings** | Настройки уведомлений |
| **/help** | Справка по командам |

### Управление уведомлениями

1. В приложении перейдите в **Integrations → Telegram**
2. Переключайте уведомления для:
   - **Alerts** — кастомные фильтры
   - **Signals** — оценки сигналов
   - **Whale tracking** — движения китов
3. Выберите, в какой чат отправлять сообщения (личные сообщения или канал)

### Решение проблем Telegram

**Бот не отвечает**
- Убедитесь, что вы подписаны на @erkescan_bot
- Нажмите /start для инициализации
- Если проблема остаётся, отключите и подключите заново

**Не приходят алерты**
1. Проверьте, включены ли уведомления в **Settings** бота
2. Проверьте, что в приложении установлены активные алерты
3. Проверьте, что Telegram не отключил уведомления от бота в системных настройках

**Сообщения с задержкой**
- ErkeScan отправляет алерты в реальном времени, задержка обычно < 1 сек
- Если задержка больше, это может быть проблема сети или Telegram API
- Попробуйте перезагрузить приложение

## Подключение Discord

Доступ через Discord предоставляется исключительно членам нашего закрытого сообщества. Если вы являетесь участником, подключите Discord-аккаунт для активации доступа.

### Шаги подключения

**Шаг 1: Перейдите в Integrations**

В приложении ErkeScan откройте раздел **Integrations** в боковом меню.

**Шаг 2: Нажмите "Connect Discord"**

Нажмите кнопку **Connect Discord**.

**Шаг 3: Авторизуйтесь в Discord**

Вас перенаправит на страницу авторизации Discord. Нажмите **Authorize**.

**Шаг 4: Готово**

Ваш аккаунт привязан. Синхронизация доступа может занять несколько минут.

### Решение проблем Discord

**Доступ не активируется**
- Убедитесь, что подключаете Discord-аккаунт, связанный с вашим участием в сообществе
- Попробуйте отключиться и переподключиться

**Остались вопросы?** Напишите в [@erkescan\_support](https://t.me/erkescan_support).

## Управление интеграциями

### Отключение интеграции

Если вы хотите отключить Telegram или Discord:

1. Перейдите в **Integrations**
2. Нажмите **Disconnect** рядом с нужной платформой
3. Подтвердите

После отключения вы перестанете получать алерты через эту платформу, но сохраните их в приложении.

### Переключение между несколькими каналами

Вы можете настроить несколько каналов для различных типов алертов:

- **Основные алерты** → #main-alerts
- **High-confidence сигналы** → #signals-hc
- **Whale tracking** → #whale-alerts

Это помогает организовать информацию и не пропустить важное.

## Лучшие практики

### Для Telegram
- Используйте отдельный чат для ErkeScan алертов, чтобы они не теряись в общении
- Включите звуковые уведомления для важных алертов
- Проверяйте /alerts регулярно для обзора активных фильтров

### Для Discord
- Создайте отдельный канал #erkescan-alerts для чистоты
- Настройте role mentions для оповещения @traders
- Используйте thread replies для обсуждения алертов
