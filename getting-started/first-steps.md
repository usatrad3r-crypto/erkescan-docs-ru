# Первые шаги

Ваш аккаунт создан, подписка активна, интеграции подключены. Теперь начните использовать ErkeScan. В этом руководстве показаны основные шаги для новичков.

## Шаг 1: Откройте скринер

Перейдите на вкладку **Screener** в главном меню.

Вы увидите таблицу с парами, отсортированную по объёму. Каждая пара показывает:
- Цену (Price)
- Изменение за 24 часа (Change 24h)
- Объём (Volume)
- OI и другие метрики

## Шаг 2: Изучите метрики

Нажмите на заголовок столбца, чтобы отсортировать по этой метрике. Например:

| Метрика | Когда использовать |
|---------|-------------------|
| **Volume** | Найти пары с высокой активностью |
| **OI (Open Interest)** | Отследить позиции трейдеров |
| **Funding Rate** | Изучить настроение (positive = bullish) |
| **Volatility** | Найти пары с хорошей амплитудой |
| **CVD** | Понять направление денег (аккумуляция/дистрибьюция) |
| **Liquidations** | Смотреть уровни жидкости |

## Шаг 3: Добавьте пару в избранное

Если вы нашли интересующую вас пару:

1. Нажмите на название пары (например, BTC/USDT)
2. Нажмите звёздочку (★) рядом с названием
3. Пара появится в разделе **Favorites** для быстрого доступа

## Шаг 4: Посмотрите на сканер сигналов

Откройте вкладку **Signal Scanner**.

Здесь вы увидите список пар с автоматическими сигналами:
- **Grade** (оценка) — от A (лучший сигнал) до F (худший)
- **Direction** — Long, Short или Neutral
- **Confidence** — уверенность алгоритма
- **Timeframe** — на каком интервале сигнал появился

### Как читать grades

- **A** — Очень сильный сигнал, множество положительных факторов
- **B** — Хороший сигнал, большинство факторов положительны
- **C** — Нейтральный сигнал, смешанные факторы
- **D** — Слабый сигнал, большинство факторов отрицательны
- **E, F** — Очень слабые сигналы или шум, игнорируйте

Фильтруйте по grades **A-B-C** и игнорируйте D, E, F.

## Шаг 5: Проанализируйте пару в графиках

Найдите интересующую вас пару и нажмите на неё. Откроется страница пары с графиком.

### На странице пары вы увидите:

1. **Цена и свечи** — классический график OHLC
2. **Индикаторы** — CVD, OI, funding rates на графике
3. **Метрики слева** — численные значения всех показателей
4. **Временные интервалы** (5m, 15m, 1h, 4h, daily) — переключайтесь между ними

### Как анализировать:

- **CVD (Cumulative Delta Volume)** — растёт = аккумуляция, падает = дистрибьюция
- **OI (Open Interest)** — растёт = добавляются позиции, падает = закрываются
- **Funding Rate** — положительный = длинных позиций больше (bullish), отрицательный = коротких (bearish)
- **Volatility** — высокая = ожидается движение

## Шаг 6: Создайте первый кастомный алерт

Алерты — это основной инструмент ErkeScan. Они уведомляют вас, когда определённые условия выполняются.

1. Перейдите в **Alerts**
2. Нажмите **+ Create Alert** или **New Alert**
3. Заполните фильтры:
   - **Pair** — выберите одну пару или "All" для всех
   - **Metric** — выберите метрику (Volume, OI, Volatility и т.д.)
   - **Condition** — выберите условие (>, <, =, increases, decreases и т.д.)
   - **Value** — введите значение
4. Нажмите **Save Alert**

### Пример алерта для новичка:

```
Pair: All
Metric: Volume 1h
Condition: > 2000000
Action: Notify via Telegram
```

Этот алерт сработает когда объём за час превысит 2 млн.

## Шаг 7: Отследите киты

Откройте вкладку **Whale Tracking**.

Здесь вы видите:
- Крупные переводы между кошельками
- Накопление в известных адресах
- Распределение из whale wallets

Если кит начал накапливать пару, которая вас интересует, это часто означает, что позже произойдёт ценовое движение.

## Шаг 8: Используйте Spaghetti Charts

Это продвинутый инструмент, но он полезен для поиска коррелированных пар.

1. Перейдите в **Spaghetti Charts**
2. Выберите несколько пар
3. На одном графике вы видите их цены вместе

Это помогает найти пары, которые движутся вместе (часто это разные торговые пары одного актива).

## Советы для новичков

### Не переусложняйте

На начальном этапе используйте только:
- Скринер (отсортируйте по Volume или Volatility)
- Signal Scanner (ловите A-grade сигналы)
- Один-два простых алерта

Постепенно добавляйте сложность по мере опыта.

### Следите за фундаментальными метриками

- **Volume** — подтверждение интереса трейдеров
- **OI + Funding rate** — определяют лонг/шорт баланс
- **CVD** — показывает накопление или распределение

### Используйте несколько timeframes

Не смотрите только на 5m. Проверяйте 1h и 4h для контекста.
- 5m/15m — микродвижения для скальпинга
- 1h/4h — main trend для swing
- daily — долгосрочное направление

### Начните с высокого качества

Фильтруйте по:
- Volume > 1M (достаточно ликвидности)
- OI > 100M (достаточно позиций)
- Volatility > 3% (есть движение)
- Grade A-B (качественные сигналы)

### Отслеживайте пару несколько дней

Не прыгайте на каждый алерт. Следите за парой в течение нескольких дней, чтобы понять её поведение.

## Частые ошибки новичков

### Много алертов сразу

Начните с 2-3 алертов. Когда вы поймёте, как они работают, добавьте больше.

### Фильтры слишком узкие

Если вы установите очень жёсткие условия, алерты никогда не сработают. Начните с более мягких условий.

### Игнорирование timeframes

Не смотрите только на 5m. Контекст с более высоких timeframes важен для правильного анализа.

### Доверие только Volume

Volume — это хорошо, но всегда проверяйте OI, Funding rate и CVD. Объём без позиций может быть обманчив.
