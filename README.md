# Добро пожаловать в ErkeScan

ErkeScan — профессиональная платформа аналитики крипторынка, разработанная для фьючерсных трейдеров, которые хотят оставаться впереди рынка.

## Что вы получаете

### Скринер в реальном времени
Мониторьте 3000+ торговых пар одновременно с помощью нашего многофункционального скринера. Сортируйте по 30+ метрикам, отслеживайте волатильность, объёмы и открытые позиции в реальном времени.

### Сканер сигналов
Автоматический анализ сигналов с оценкой качества (A-F) и направлением. Алгоритм обучается на истории рынка и даёт вам готовый инструмент для принятия решений.

### Пользовательские алерты
Настройте неограниченное количество алертов на основе любых параметров. Получайте уведомления в реальном времени через Telegram.

### Графики и аналитика
Анализируйте данные с 5 временными интервалами. CVD, funding rates, OI, volume profile и другие метрики в одной платформе.

### Spaghetti Charts
Визуализируйте множество монет на одном графике. Быстро находите паттерны и коррелированные движения.

## Поддерживаемые биржи

- **Binance** — фьючерсы и спот
- **Bybit** — фьючерсы и спот
- **Hyperliquid** — фьючерсы

Данные обновляются каждые несколько секунд. Более 30 показателей рассчитываются на каждой паре.
